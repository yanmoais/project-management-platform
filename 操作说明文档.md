# 项目管理平台操作说明文档

## 项目概述

项目管理平台是一个基于Vue 3 + FastAPI的现代化全栈Web应用，提供项目管理、自动化测试、质量管控、生产管理等全方位功能。

## 技术栈

### 前端
- **Vue 3** - 前端框架
- **Element Plus** - UI组件库
- **Vue Router** - 路由管理
- **Pinia** - 状态管理
- **Vite** - 构建工具
- **TypeScript** - 类型检查

### 后端
- **FastAPI** - Web框架
- **SQLAlchemy** - ORM
- **Celery** - 异步任务队列
- **Eventlet** - 异步I/O
- **MySQL** - 数据库

## 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- MySQL 5.7+

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yanmoais/project-management-platform.git
   cd project-management-platform
   ```

2. **安装Python依赖**
   ```bash
   # 创建虚拟环境
   python -m venv venv
   
   # 激活虚拟环境
   # Windows
   venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   
   # 安装依赖
   pip install -r backend_fastapi/requirements.txt
   ```

3. **安装前端依赖**
   ```bash
   npm install
   ```

4. **数据库配置**
   - 导入 `create.sql` 到MySQL数据库
   - 修改 `backend_fastapi/config.py` 中的数据库连接配置

5. **启动服务**
   ```bash
   # 使用一键启动脚本
   start_all.bat
   
   # 或者分别启动
   # 后端
   cd backend_fastapi
   uvicorn main:app --host 0.0.0.0 --port 5000 --reload
   
   # 前端
   npm run dev
   ```

## 开发指南

### 前端开发
1. **组件开发**
   - 使用Vue 3 Composition API
   - 遵循Element Plus设计规范
   - 组件命名使用PascalCase

2. **API调用**
   ```javascript
   import { getProjectList } from '@/api/ProjectMgt'
   
   const { data, error } = await getProjectList({
     page: 1,
     pageSize: 10
   })
   ```

3. **状态管理**
   ```javascript
   import { useProjectStore } from '@/store/ProjectMgt'
   
   const projectStore = useProjectStore()
   projectStore.setProjects(data)
   ```

### 后端开发
1. **API路由**
   ```python
   from fastapi import APIRouter
   
   router = APIRouter()
   
   @router.get("/projects")
   async def get_projects():
       return {"projects": []}
   ```

2. **数据库操作**
   ```python
   from sqlalchemy.orm import Session
   from backend_fastapi.db.session import get_db
   
   def get_project_by_id(db: Session, project_id: int):
       return db.query(Project).filter(Project.id == project_id).first()
   ```

3. **Celery任务**
   ```python
   from celery import shared_task
   
   @shared_task
   def send_notification(user_id: int, message: str):
       # 发送通知逻辑
       pass
   ```

## 部署指南

### 开发环境
1. 使用 `start_all.bat` 启动所有服务
2. 前端访问: http://localhost:5173
3. 后端API: http://localhost:5000

### 生产环境
1. **构建前端**
   ```bash
   npm run build
   ```

2. **启动后端**
   ```bash
   uvicorn backend_fastapi.main:app --host 0.0.0.0 --port 5000
   ```

3. **配置Nginx反向代理**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       
       location / {
           root /path/to/dist;
           try_files $uri $uri/ /index.html;
       }
       
       location /api {
           proxy_pass http://localhost:5000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
       }
   }
   ```

## 常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr :5000
# 杀死占用进程
taskkill /PID <进程ID> /F
```

### 2. 数据库连接失败
- 检查MySQL服务是否启动
- 确认数据库配置正确
- 检查防火墙设置

### 3. Celery任务不执行
- 确认Redis服务运行
- 检查Celery Worker是否启动
- 查看Celery日志
