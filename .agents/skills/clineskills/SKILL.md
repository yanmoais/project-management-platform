# 技能规则文档 (Skills Rules)

# 代码注释规范
1. 所有Python代码的注释必须使用中文
2. 每个函数上方要有中文说明这个函数是做什么的
3. 复杂的代码行旁边要有中文解释

# 调用与逻辑校验
1、引入新方法调用时，必须严格检查调用方与被调用方的参数、返回值是否一致。
2、确保被调用的方法已正确生成，且方法体内部逻辑正确无误，不得出现空引用或逻辑漏洞。
3、在编写调用逻辑的同时，必须同步确认被调用方法的具体实现。

# 数据库操作规范
1、进行数据库操作前，必须查询数据库表结构和 `models.py` 进行双重 Check，确保字段结构无误。
2、严格按照现有表结构进行字段映射，严禁私自臆造或修改字段名。
3、代码中的字段命名必须与数据库表字段完全匹配，防止因字段名不一致导致的运行错误。
